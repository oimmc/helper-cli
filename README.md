# helper-cli

> Usage:

``` bash

npm install helper-cli -g

helper <command> [options]

```


> Options:

	-v --version       output the version number

	-h, --help         output usage information


> Commands:

	init|i <app-name>  init a new project


## Commands:

``` bash
# init a new project
init|i <app-name>

```
